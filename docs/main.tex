\begin{abstract}
Makalah ini menyajikan analisis komparatif antara ASCON-128 (Standar NIST LWC) dan AES-128-GCM pada simulasi lingkungan terbatas. Kami berfokus pada latensi CPU, penggunaan memori, dan overhead kode untuk aplikasi kunci sepeda pintar. Hasil pengujian menunjukkan bahwa ASCON-128 memiliki keunggulan dalam penggunaan memori yang lebih rendah, menjadikannya kandidat kuat untuk perangkat IoT dengan sumber daya sangat terbatas, meskipun AES-128-GCM unggul dalam throughput pada perangkat dengan akselerasi perangkat keras.
\end{abstract}
